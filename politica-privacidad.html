Política de Privacidad

Ultima actualización: 23 de febrero de 2026

En cumplimiento del Reglamento (UE) 2016/679 General de Protección de Datos (RGPD) y la Ley Orgánica 3/2018 de Protección de Datos Personales y garantía de los derechos digitales (LOPDGDD), te informamos sobre el tratamiento de tus datos personales al visitar este sitio web.

1. Responsable del tratamiento

  • Nombre: Txema González Balseiro
  • Sitio web: txemagonzalez.com
  • Email: txema [arroba] txemagonzalez.com

2. Datos personales que tratamos

Este sitio web es un blog estático alojado en GitHub Pages. No recogemos datos personales a través de formularios, no requiere registro y no utiliza cookies de terceros.

2.1. Estadísticas de uso (localStorage)

Utilizamos localStorage del navegador para almacenar un contador anónimo de visitas por artículo. Estos datos:

  • Se almacenan exclusivamente en tu navegador. No se envían a ningún servidor.
  • No contienen información personal identificable (ni IP, ni nombre, ni email).
  • Solo registran: identificador del artículo, número de visitas y fecha de última visita.
  • Se eliminan automáticamente transcurridos 13 meses desde la última visita.
  • Puedes eliminarlos manualmente en cualquier momento (ver Política de Cookies).

Base legal: interés legítimo (art. 6.1.f RGPD) y exención de la AEPD para analíticas propias no intrusivas.

2.2. Logs del servidor (GitHub Pages)

El alojamiento en GitHub Pages registra automáticamente datos técnicos estándar (dirección IP, tipo de navegador, fecha y hora de acceso). Estos datos son procesados por GitHub Inc. como encargado del tratamiento, según su Política de Privacidad y su acuerdo de procesamiento de datos (DPA).

Base legal: interés legítimo en el funcionamiento técnico del sitio web (art. 6.1.f RGPD).

3. Tipografías

Las fuentes tipográficas (Poppins, Lora, Abril Fatface) están alojadas localmente en este servidor. No se realiza ninguna conexión a servidores externos de Google para cargar tipografías.

4. Destinatarios

No cedemos tus datos personales a terceros, salvo:

  • GitHub Inc. (alojamiento web): procesa logs técnicos del servidor. Certificada bajo el EU-US Data Privacy Framework.

5. Transferencias internacionales

GitHub Inc. tiene sede en Estados Unidos y está certificada bajo el EU-US Data Privacy Framework, que garantiza un nivel de protección adecuado según la Decisión de Adecuación de la Comisión Europea de 10 de julio de 2023.

6. Plazos de conservación

  • localStorage: máximo 13 meses desde la última visita (purgado automático).
  • Logs de GitHub Pages: según la política de retención de GitHub (normalmente 90 días).

7. Derechos del usuario

Puedes ejercer los siguientes derechos escribiendo a nuestro email de contacto:

  • Acceso: solicitar qué datos tenemos sobre ti.
  • Rectificación: corregir datos inexactos.
  • Supresión: solicitar la eliminación de tus datos.
  • Limitación: solicitar que restrinjamos el uso de tus datos.
  • Portabilidad: recibir tus datos en formato estructurado.
  • Oposición: oponerte al tratamiento de tus datos.

Dado que los datos de localStorage residen exclusivamente en tu navegador, puedes ejercer tu derecho de supresión directamente borrando los datos del navegador (ver Política de Cookies).

También tienes derecho a presentar una reclamación ante la Agencia Española de Protección de Datos (www.aepd.es).

8. Menores

Este sitio web no está dirigido a menores de 14 años. No recopilamos conscientemente datos personales de menores.

9. Enlaces a terceros

Este sitio contiene enlaces a plataformas externas (GitHub, LinkedIn, Twitter, Buy Me a Coffee). Al hacer clic en estos enlaces, abandonas este sitio web y quedas sujeto a las políticas de privacidad de dichas plataformas.

10. Modificaciones

Nos reservamos el derecho de modificar esta política de privacidad para adaptarla a novedades legislativas o cambios en el sitio web. La fecha de última actualización se indica al inicio de esta página.
